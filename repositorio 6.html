Repositorio de Elementos XHTML

<img> Imagen
Descripción General

Se emplea para incluir imágenes en los documentos[cite: 243].

Tipo de elementoEn línea y etiqueta vacía [cite: 243]
Atributos comunesBásicos, i18n y eventos [cite: 243]
Atributos Específicos
src = "url"Indica la URL de la imagen que se muestra (Requerido)[cite: 243, 244].
alt = "texto"Descripción corta de la imagen (Requerido). Máximo 1024 caracteres[cite: 243, 246].
longdesc = "url"Indica una URL con una descripción más detallada de la imagen[cite: 243].
name = "texto"Nombre del elemento imagen[cite: 243].
heightIndica la altura con la que se debe mostrar la imagen (píxeles o %)[cite: 243, 281].
widthIndica la anchura con la que se debe mostrar la imagen (píxeles o %)[cite: 243, 281].
<map> Mapa de imagen
Descripción General

Se emplea para definir mapas de imagen del lado del cliente[cite: 325].

Tipo de elementoBloque y en línea [cite: 325]
Atributos comunesBásicos, i18n y eventos [cite: 325]
Atributos Específicos
name = "texto"Nombre que identifica de forma única al mapa definido (Obligatorio)[cite: 325].
<area> Área de un mapa de imagen
Descripción General

Se emplea para definir las distintas áreas que forman un mapa de imagen[cite: 325].

Tipo de elementoEtiqueta vacía [cite: 325]
Atributos comunesBásicos, i18n, eventos y foco [cite: 325]
Atributos Específicos
href = "url"URL a la que se accede al pinchar sobre el área[cite: 325].
nohrefSe emplea para las áreas que no son seleccionables[cite: 325].
shapeTipo de área: default, rect, circle o poly[cite: 325].
coordsCoordenadas del área. Formato depende de shape (X1,Y1,X2,Y2 para rect; X,Y,R para circle)[cite: 325].
<object> Objeto
Descripción General

Se emplea para embeber objetos complejos (Java, vídeos, Flash) en los documentos[cite: 351, 356].

Tipo de elementoBloque y en línea [cite: 356]
Atributos comunesBásicos, i18n y eventos [cite: 356]
Atributos Específicos
data = "url"Indica la URL de los datos o recurso que utiliza el objeto[cite: 356].
typeIndica el tipo de contenido de los datos (MIME type)[cite: 356, 358].
height / widthIndica las dimensiones con las que se debe mostrar el objeto[cite: 356].
classid, codebase, codetypeInformación específica que depende del tipo de objeto[cite: 356].

<embed> Embeber objetos (No estándar)
Descripción General

Se emplea para embeber objetos. No es parte del estándar oficial XHTML, pero es útil para compatibilidad con vídeos Flash[cite: 391, 393].

Tipo de elementoBloque [cite: 393]
Atributos comunesBásicos, i18n y eventos [cite: 393]
Atributos Específicos
src = "url"Indica la URL del archivo u objeto que se incluye[cite: 393].
typeIndica el tipo de contenido (flash, quicktime, java, etc.)[cite: 393].
height / widthDimensiones del objeto[cite: 393].
